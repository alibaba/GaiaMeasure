# 阴影

<img src="https://img.alicdn.com/imgextra/i4/O1CN01Ry9My61VLKlgpJVn6_!!6000000002636-2-tps-628-498.png" style="zoom:50%;" />

阴影（box-shadow）属性包括：阴影颜色、x偏移量、y偏移量、阴影模糊半径、阴影扩散半径

* 颜色：阴影颜色
* X：x偏移量
* Y：y偏移量
* 模糊：阴影模糊半径
* 扩展：阴影扩散半径
