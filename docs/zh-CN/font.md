# 字符

对于文本类型的图层，会单独展示字符相关属性，包括字体、字重、行高、字号、颜色、对齐方式等

* 字体 （font-family）

* 字重（font-weight），对应关系如下：

  | 数值 | 名称                |
  | ---- | ------------------- |
  | 300  | 细体（Light）       |
  | 400  | 常规体（Regular）   |
  | 500  | 中黑体（Medium）    |
  | 600  | 中粗体（Semi Bold） |
  | 700  | 粗体（Bold）        |

* 行高（line-height）

* 字号（font-size）

* 颜色（color）

* 水平对齐方式（text-align）
