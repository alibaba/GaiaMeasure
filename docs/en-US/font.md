For text-type layers, character-related properties are displayed separately, including font, font weight, line height, font size, color, alignment, etc.

* Font-weight, the corresponding relationship is as follows:

   | Value | Name |
   | ---- | ------------------- |
   | 300 | Light |
   | 400 | Regular |
   | 500 | Medium |
   | 600 | Semi Bold |
   | 700 | Bold |

* Font size (font-size)
